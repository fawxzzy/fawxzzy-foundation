# Vercel Setup

## Planned project

- Team slug: `fawxzzy`
- Project name: `fawxzzy-foundation`
- Output directory: `apps/console/public`
- Build command: `pnpm build`

## Link locally

```bash
vercel link --yes --project fawxzzy-foundation --scope fawxzzy
```

## Deploy

```bash
vercel deploy
```

For production:

```bash
vercel deploy --prod
```

## Vercel config

`vercel.json` is configured for a static console generated by the build command.

## After first deployment

Update `data/projects.json`:

- set `foundation.vercel.exists` to `true`,
- add the Vercel project ID if available,
- run `pnpm build && pnpm verify:local`,
- commit the registry update.
